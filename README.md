# Flexbox Pricing Table Project

The **Flexbox Pricing Table Project** is a responsive pricing section example built using CSS **Flexbox**.  
It demonstrates the use of modern layout techniques to create clean, flexible, and mobile‑friendly UI components.

---

## 🌐 Live Demo

Try it here:  
👉 https://bahadirbektore.github.io/flexbox-pricing--table--project/

---

## 🚀 Project Overview

This project includes:

- A responsive pricing table layout  
- Clean, modern component design  
- Fully flexible layout using **CSS Flexbox**  
- Works across desktop and mobile screens  

---

## 🛠 Technologies Used

This project was built using:

- **HTML5** – Semantic markup  
- **CSS3** – Flexbox for layout  
- **Responsive design** principles
